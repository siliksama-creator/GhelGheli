# Admin Panel

```bash
npm install
VITE_API_BASE=http://localhost:4000 npm run dev
```

پنل کاملاً RTL و فارسی است و همه بخش‌های اصلی مدیریت کارت، کد، جوایز، لیگ، کاربران، چت، پشتیبانی، اطلاعیه و ادمین‌ها را پوشش می‌دهد.

## ورود تست

بعد از اجرای `npm run seed:admin` در backend:

```text
Username: Admin
Password: admin
```
